 🚀 ChainEscrow : A Blockchain-Based Freelance Payment System

ChainEscrow is a decentralized application (DApp) that enables secure and trustless freelance payments using Ethereum smart contracts. It implements an escrow mechanism where funds are locked until work is completed, ensuring fairness for both clients and freelancers.

----------------------------------------------------------------------------------------------------

## 📌 Features

- 🔐 Secure escrow-based payments using smart contracts  
- 💸 Funds locked until client approval  
- 🦊 MetaMask wallet integration  
- 🔗 Transparent blockchain transactions  
- 📊 Project tracking with status (Pending / Completed)  

---------------------------------------------------------------------------------------------------

## 🛠️ Tech Stack

- **Blockchain:** Ethereum (Localhost - Hardhat)  
- **Smart Contracts:** Solidity  
- **Frontend:** HTML, CSS, JavaScript  
- **Web3 Integration:** Ethers.js  
- **Wallet:** MetaMask  
- **Development Framework:** Hardhat  

----------------------------------------------------------------------------------------------------

## ⚙️ How It Works

1. Client connects wallet using MetaMask  
2. Client enters freelancer address and payment amount  
3. Funds are sent and locked in the smart contract (Escrow)  
4. Project is created and stored on blockchain  
5. After work completion, client releases payment  
6. Smart contract transfers funds to freelancer
